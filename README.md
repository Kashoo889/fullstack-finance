# Kashif Hisaab Kitaab - Finance Dashboard

Complete MERN stack application for managing financial accounts and transactions.

## Project Structure

```
Main-folder/
├── frontend/          # React + TypeScript + Tailwind CSS frontend
├── backend/           # Node.js + Express + MongoDB backend
└── README.md          # This file
```

## Quick Start

### Backend Setup

```bash
cd backend
npm install
# Create .env file (see backend/ENV_SETUP.md)
npm run dev
```

Backend runs on `http://localhost:5000`

### Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Frontend runs on `http://localhost:8080`

## Features

- ✅ Saudi Hisaab Kitaab - Track SAR transactions with PKR conversions
- ✅ Special Hisaab Kitaab - Manage user balances (Online/Cash)
- ✅ Pakistani Hisaab Kitaab - Manage traders with multiple bank accounts
- ✅ Server-side balance calculations
- ✅ Running balance tracking
- ✅ RESTful API
- ✅ Responsive design
